Caveman 技能

不止是说话方式。更多强大功能。

📝

caveman-commit

简洁的提交消息。Conventional Commits。≤50 字符的主题。原因重于内容。

/caveman-commit
🔍

caveman-review

单行 PR 评论：L42: 🔴 bug: user null. Add guard. 没有废话。

/caveman-review
🗜️

caveman-compress

caveman 让 Claude 说话时使用更少的 token。Compress 让 Claude 阅读时使用更少的 token。每会话减少约 46% 输入 token。

/caveman:compress CLAUDE.md
❓

caveman-help

快速参考卡。所有模式、技能、命令，一个命令即可获取。

/caveman-help
